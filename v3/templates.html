Basic layout names

Names of Basics layouts se systoqt ot 3 chasti.

W pyrwata e posocheno systoqnieto na osnownite horizontalni poleta (header1,header2,footer2,footer1,copyright)

W vtorata e posocheno systoqnieto i razpolojenieto na vynshnite koloni i obshivkata (sidebar1,sidebar2,wrapper)

W tretata e posocheno systoqnieto i razpolojenieto na vytreshnite koloni i contenta (sidebar3,sidebar4,content)
#header1 #header2 #footer2 #footer1 #copyright _ position1 position2 position3 position4 position5 position6
f (fixed) 2 (static) 2 (static) 1 (static) f (fixed) _ 1 (sidebar1) w (wrapper) 2 (sidebar2) 3 (sidebar3) c (content) 4 (sidebar4)
1 (static) x (none) x (none) x (none) _ 2 (sidebar2) 1 (sidebar1) w (wrapper) 4 (sidebar4) 3 (sidebar3) c (content)
x (none) _ w (wrapper) 2 (sidebar2) 1 (sidebar1) c (content) 4 (sidebar4) 3 (sidebar3)
_ x (none) x (none) x (none) x (none) x (none) x (none)
In position1, position2 or position3 must be present w and elements 1,2,w can not be duplicated.
In position4, position5 or position6 must be present c and elements 3,4,c can not be duplicated.
x means that the element is not present in this layout.

Theme colors

the theme colors is set as the name of the css file (Example: bgrey.css). Read more about Themes colors.

Accent colors

the Accent colors is set as the name of the css file plus uppercase A (Example: indigoA.css). Read more about Accent colors.
