Position

The position Classes
The position property specifies the type of positioning method used for an element.

There are four different position classes:
.static
.relative
.fixed
.absolute

absolute_top_left
absolute_top_center
absolute_top_right
absolute_left_center
absolute_center
absolute_right_center
absolute_bottom_left
absolute_bottom_center
absolute_bottom_right
